ขั้นตอนที่ 2 ประเมินโอกาสการเกิดอันตราย

คำชี้แจง : โดยใส่คะแนนในช่องโอกาส ตามเกณฑ์การพิจารณา

โอกาสการเกิดอันตราย

เกณฑ์การประเมิน/พิจารณา

โอกาส

1. จำนวนคนที่สัมผัส

0 = ไม่เกี่ยวข้อง
1 = 1 – 5 คน
2 = 6 - 10 คน
3 = >10 คน

 

{{number1}}

2. ความถี่และระยะเวลาที่สัมผัส

0 = ไม่เกี่ยวข้อง
1 = สัมผัสน้อยกว่า 10 ชม./สัปดาห์
2 = สัมผัสระหว่าง 10-30 ชม./สัปดาห์
3 = สัมผัสมากกว่า30 ชม./สัปดาห์

 

{{number2}}

3. การตรวจวัดสภาพแวดล้อมในการทำงาน

0 = ไม่เกี่ยวข้อง
1 = มีการตรวจวัดและเป็นไปตามที่มาตรฐานกฎหมายกำหนด
2 = มีการตรวจวัดแต่ค่าเกินมาตรฐานกฎหมาย
3 = ไม่มีการตรวจวัด

 

{{number3}}

4. มีขั้นตอน/วิธีการปฏิบัติที่ได้มาตรฐาน

0 = ไม่เกี่ยวข้อง
1 = มีขั้นตอน/วิธีการเป็นลายลักษณ์อักษรและเหมาะสมกับความเสี่ยง
2 = มีขั้นตอน/วิธีการปฏิบัติงานเป็นลายลักษณ์อักษร แต่ไม่เหมาะสมกับความเสี่ยง
3 = ไม่มีขั้นตอน/วิธีการปฏิบัติงานเป็นลายลักษณ์อักษร

 

{{number4}}

5. มีการฝึกอบรมขั้นตอน/วิธีการปฏิบัติอย่างมีประสิทธิภาพ

0 = ไม่เกี่ยวข้อง
1 = มีการกำหนดความต้องการฝึกอบรมและมีการฝึกอบรม
2 = มีการฝึกอบรมแต่ไม่มีการกำหนดความต้องการการฝึกอบรมหรือมีกำหนดความต้องการฝึกอบรมแต่ไม่มีการฝึกอบรม
3 = ไม่มีกำหนดความต้องการฝึกอบรมและการฝึกอบรม

 

{{number5}}

6. การควบคุมการปฏิบัติตามขั้นตอน/วิธีการปฏิบัติที่ได้มาตรฐาน

0 = ไม่เกี่ยวข้อง
1 = มีการควบคุมการปฏิบัติอย่างต่อเนื่อง
2 = มีการควบคุมการปฏิบัติแต่ไม่ต่อเนื่อง
3 = ไม่มีการควบคุมการปฏิบัติ

 

{{number6}}

7. อุปกรณ์ป้องกันอันตรายส่วนบุคคล

0 = ไม่เกี่ยวข้อง
1 = มีอย่างเหมาะสม
3 = ไม่มี, มีแต่ไม่เหมาะสม

 

{{number7}}

8. เครื่องมือ,เครื่องจักร,อุปกรณ์,อุปกรณ์ความปลอดภัย

0 = ไม่เกี่ยวข้อง
1 = ความปลอดภัยอย่างเหมาะสม, มีการออกแบบให้มีอุปกรณ์คุ้มครอง
3 = ไม่มี, มีแต่ไม่เหมาะสม

 

{{number8}}

9. การบำรุงรักษาอุปกรณ์ เครื่องมือ เครื่องจักร

0 = ไม่เกี่ยวข้อง
1 = มีการบำรุงรักษาและมีการบันทึกอย่างต่อเนื่อง
2 = มีการบำรุงรักษาแต่ไม่มีการบันทึก
3 = ไม่มีการบำรุงรักษา

 

{{number9}}

10. การเตือนอันตราย

0 = ไม่เกี่ยวข้อง
1 = มีการเตือนอันตรายและเหมาะสมกับลักษณะความเสี่ยง
2 = มีการเตือนอันตรายแต่ไม่เป็นมาตรฐานเดียวกัน
3 = ไม่มีการเตือนอันตราย

 

{{number10}}

 

     

ขั้นตอนที่ 3 การประเมินความรุนแรง

คำชี้แจง : ใส่ค่าความรุนแรงในช่องว่าง ตามเกณฑ์การพิจารณาของระดับความรุนแรงนั้นๆ

ความรุนแรง

1. รุนแรงมาก

คน : แขนขาด ขาขาด นิ้วขาด กระดูกแตกรุนแรง สารพิษ ตาย

ทรัพย์สิน : เสียหายมูลค่ามากกว่า 100,000 บาท

 

{{violence}}

2. รุนแรงปานกลาง

คน : ผิวเปิด ถูกบาดเป็นแผลลึก ถูกไฟไหม้ อาเจียน กระดูกแตกเล็กน้อย แขนขาผิดปกติ

ทรัพย์สิน : ความเสียหายมูลค่าตั้งแต่ 5,000-100,000 บาท

3. รุนแรงน้อย

คน : ผิวหนังชั้นบนได้รับบาดเจ็บ เช่นถูกบาด ขูด ระคายเคืองเนื่องจากฝุ่น รำคาญเนื่องจากระคายเคือง อึดอัด ไม่สบาย

ทรัพย์สิน : ความสูญเสียมูลค่าไม่เกิน 5,000 บาท
